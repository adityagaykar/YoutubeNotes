Q. How to run :

1. git clone https://iiith.visualstudio.com/_git/MyFirstProject
[ Enter your credentials and clone the project to local repo ]

2. -> Open google chrome, type url- chrome://extensions
   -> Check 'Developer mode'
   -> click 'Load unpacked extension' > choose 'MyFirstProject/Youtube_Notes'
   -> Go to a any youtube video. Start making notes from your video.


Q. Features :

1. Play/Pause vidoe from the extension
2. Make notes at specific timestamps
3. View all notes of the video on the timeline. Traverse through the timeline by clicking on timestamps.
4. View All videos list
5. Bing Search embedded in extension.
